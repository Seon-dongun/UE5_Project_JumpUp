## UE5_Project_JumpUp

- 언리얼 엔진5로 제작한 플랫폼 게임 JumpUp 입니다  
- 점프를 통해 장애물을 극복하여 목표 지점까지 도달하는 게임입니다 


- 해당 프로젝트는 블루프린트만을 활용하여 제작하였습니다 


- Asset 폴더에 사용된 애셋들을 담아두었습니다  

<br>

### [조작법]  
이동 : WASD / 방향키  
점프 : Space 바
